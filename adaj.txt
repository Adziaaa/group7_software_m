✅ List of Important Domain Classes (Initial Set)
| **Domain Class**                                               | **Responsibility**                    | **What to Do**                                         |
| -------------------------------------------------------------- | ------------------------------------- | ------------------------------------------------------ |
| **Application (Main)**                                         | Starting point of application         | Launch the drawing app, initialize UI and tools        |
| **Controller / Action Classes** (e.g., DrawAction, SaveAction) | Handle user input events              | Respond to buttons, menus, and tool events             |
| **DrawingEditor**                                              | Coordinates tools and views           | Manage current tool, views, and drawing state          |
| **Drawing**                                                    | Domain model container for figures    | Add/remove figures, supports undo/redo                 |
| **Figure**                                                     | Represents a graphical object         | Draw shape, change position, support handles           |
| **Tool**                                                       | Reacts to user input (mouse/keyboard) | Interpret clicks/drags to create or manipulate figures |
| **Handle**                                                     | Enable figure manipulation            | Resize/move shapes from the view                       |
| **Connector**                                                  | Determine connection points           | Used by connection tools to link figures               |
| **DrawingView**                                                | Visual representation of drawing      | Show figures on screen and handle repaint              |
| **Data Transfer (Clipboard/IO)**                               | Save/load drawings                    | Handle copy, paste, open and save                      |
| **GUI Components** (e.g., Frame, Palette)                      | UI interface                          | Render menus, buttons, toolbars                        |
| **Utility Classes** (Helpers/Utils)                            | Support operations                    | Geometry handling, serialization                       |
| **Sample App Classes**                                         | Demonstrate framework usage           | Example applications to run/debug                      |
